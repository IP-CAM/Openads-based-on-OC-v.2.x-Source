save

public mixed save($filename)

Method for saving the whole OLE container (including files).
In fact, if called with an empty argument (or '-'), it saves to a temporary file and then outputs it's contents to stdout.
If a resource pointer to a stream created by fopen() is passed it will be used, but you have to close such stream by yourself.
